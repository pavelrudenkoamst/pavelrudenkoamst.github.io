# Trivia Questions Visualizer

A React-based visualization tool that displays trivia questions data from the [Open Trivia DB API](https://opentdb.com). The app provides interactive charts and filtering capabilities to explore question categories and difficulty levels.

## Features

- **Category Distribution Chart** - Bar chart showing questions by category
- **Difficulty Distribution Chart** - Pie chart displaying questions by difficulty level
- **Category Filtering** - Filter questions by selecting specific categories
- **Statistics Display** - View total questions and category counts
- **Load More Questions** - Fetch additional questions with rate limit protection
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices

## Tech Stack

- **React** - UI framework with hooks
- **Recharts** - Chart visualization library
- **Vite** - Build tool and development server
- **CSS Modules** - Scoped styling

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

The production build will be in the `dist` folder.
